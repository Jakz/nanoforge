# nanoforge

A cross-platform development project with organized structure for different build systems and components.

## Project Structure

```
nanoforge/
├── src/           # Source code files
├── libs/          # External libraries and dependencies
└── projects/      # Platform-specific project files
    ├── xcode/     # Xcode project files for macOS/iOS development
    └── msvc/      # Microsoft Visual Studio project files for Windows
```

## Directories

- **src/**: Contains the main source code for the project
- **libs/**: Contains external libraries and dependencies
- **projects/xcode/**: Contains Xcode project files and configurations for Apple platforms
- **projects/msvc/**: Contains Microsoft Visual Studio project files and configurations for Windows
